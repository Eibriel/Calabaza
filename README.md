# Calabaza
Calabaza interactive game

## Installing

* mkdir venv
* virtualenv -p python3 venv/
* . venv/bin/activate
* git clone https://github.com/Eibriel/Calabaza.git
* cd Calabaza/server
* pip -r install requirements.txt

## Running

* cd server
* ./calabaza.py runserver

## Development
